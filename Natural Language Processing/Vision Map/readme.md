# 🤖 Vision Map: 청소년 진로 상담 챗봇

## 📖 프로젝트 개요

**Vision Map**은 청소년들의 진로 고민을 들어주고 맞춤형 로드맵을 제시하기 위해 개발된 **진로 상담 챗봇** 프로젝트입니다. (청소년 데이터 공모전 출품작)

상담 데이터를 학습한 AI 챗봇을 통해 시공간의 제약 없이 청소년들에게 정서적 지지와 실질적인 진로 정보를 제공하는 것을 목표로 했습니다.

## ✨ 핵심 프로세스 및 전략

### 1. 데이터 전처리 (Data Preprocessing)
-   **데이터 구조화**: 초/중/고등 상담 기록 데이터(JSON)를 분석하기 용이한 DataFrame 형태로 변환했습니다.
-   **대화 페어링(Pairing)**: 상담사와 학생의 발화를 분리하고, 문맥이 이어지는 발화를 하나로 묶어 **[질문-답변]** 쌍을 생성했습니다.
-   **노이즈 제거**: "네", "알겠습니다"와 같이 의미 없는 짧은 답변이나 단순 호응어 등을 필터링하여 학습 데이터의 품질을 높였습니다.

### 2. 모델링 (Modeling)
-   **Model Selection**: 한국어 생성 능력과 경량화 모델의 장점을 고려하여 `pko-t5-small` 모델을 선정했습니다.
-   **Fine-tuning**: 전처리된 상담 대화 데이터셋을 사용하여 T5 모델을 미세 조정(Fine-tuning)했습니다. 제한된 자원 내에서 최적의 성능을 내기 위해 Epoch과 Learning Rate 등 하이퍼파라미터를 튜닝했습니다.

### 3. 응답 생성 (Generation)
-   **Context Awareness**: 단순히 직전 발화에만 반응하는 것이 아니라, 이전 대화의 문맥을 기억하여 더 자연스럽고 일관된 답변을 생성하도록 로직을 구성했습니다.

```
└── README.md              # 프로젝트 설명 파일
```

## 🛠️ 주요 구성 요소

-   **`t5 fine-tuning.ipynb`**: 데이터 로드, 전처리, 토크나이저 설정, 모델 학습, 그리고 챗봇 응답 생성 테스트까지의 전체 과정이 포함된 메인 노트북입니다.
-   **`From Scratch.ipynb`**: 프로젝트 초기 단계에서 다양한 접근 방식을 실험했던 코드입니다.

## 🚀 실행 방법

1.  **데이터 준비**: 청소년 상담 데이터셋(JSON)을 준비합니다.
2.  **학습 및 실행**: `t5 fine-tuning.ipynb`를 실행하여 모델을 학습시키거나, 학습된 모델을 로드하여 챗봇과 대화를 시도합니다.
