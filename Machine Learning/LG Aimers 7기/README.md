# LG Aimers 7기: 시계열 데이터를 활용한 수요 예측

## 📖 프로젝트 개요

본 프로젝트는 LG Aimers 7기 활동의 일환으로 진행된 과제이며, 시계열 데이터를 기반으로 제품 또는 서비스의 미래 수요량을 예측하는 것을 목표로 합니다.

다양한 피처 엔지니어링 기법과 시계열 예측 모델(ARIMA, XGBoost, LSTM 등)을 실험하여 예측의 정확도를 높이는 데 집중했습니다. 최종적으로는 복잡하고 비선형적인 시계열 패턴을 효과적으로 학습할 수 있는 LSTM 모델을 중심으로 솔루션을 구축했습니다.

## 🕐 개발 기간 
2025.08 ~ 2025.08 (1개월)

## ✨ 핵심 프로세스 및 전략

### 1. 데이터 분석 및 전처리 (EDA & Preprocessing)
-   학습 데이터(`train.csv`)를 분석하여 시계열 데이터의 트렌드, 계절성, 주기성 등 주요 특징을 파악했습니다.
-   결측치를 처리하고, 모델 학습에 적합한 형태로 데이터를 정제했습니다.

### 2. 피처 엔지니어링 (Feature Engineering)
-   날짜 데이터를 기반으로 풍부한 파생 변수를 생성하여 모델의 예측 성능을 극대화했습니다.
    -   **시간 기반 변수**: 요일, 월, 연도, 분기, 주(week of year) 등
    -   **이벤트 변수**: 한국 공휴일(`holidays` 라이브러리 활용) 정보 포함
    -   **가중치 적용**: 월별, 요일별, 계절별 특성을 반영하기 위해 경험적 가중치를 산정하여 변수로 활용했습니다.

### 3. 모델링 및 실험 (Modeling & Experiments)
다양한 시계열 예측 모델의 성능을 비교 분석하며 최적의 모델을 탐색했습니다.
-   **통계 모델**: ARIMA, ARIMAX
-   **트리 기반 부스팅 모델**: XGBoost (다양한 파라미터 및 교차 검증 시도)
-   **딥러NING 모델**: **LSTM (Long Short-Term Memory)**

실험 결과, 장기 의존성 및 복잡한 패턴 학습에 강점을 보이는 LSTM이 본 과제에 가장 적합하다고 판단하여 최종 모델로 발전시켰습니다.

## 🛠️ 주요 구성 요소

-   **`[공유용] MV, Cls, Reg.ipynb`**: 데이터 로딩, 전처리, 그리고 본 프로젝트의 핵심인 피처 엔지니어링 과정이 상세하게 구현되어 있습니다.
-   **`4. Final_LSTM.ipynb`**: 다양한 피처를 입력으로 받아 미래 수요를 예측하는 최종 LSTM 모델의 학습 및 추론 코드를 포함합니다.
-   **그 외 노트북 (`ARIMAX.ipynb`, `XGBoost.ipynb` 등)**: 최종 모델을 선정하기까지 시도했던 다양한 모델링 및 실험 과정을 기록한 파일들입니다.

## 🚀 실행 방법

1.  **환경 설정**: 노트북 상단에 명시된 필요 라이브러리(`pandas`, `numpy`, `holidays`, `tensorflow` or `pytorch` 등)를 설치합니다.
2.  **데이터 준비**: 지정된 경로에 `train.csv`와 같은 학습 데이터를 준비합니다.
3.  **피처 엔지니어링 실행**: `[공유용] MV, Cls, Reg.ipynb` 또는 각 모델별 노트북의 전처리 부분을 실행하여 모델 학습에 필요한 데이터를 생성합니다.
4.  **모델 학습 및 추론**: `4. Final_LSTM.ipynb`를 실행하여 최종 모델을 학습하고 결과를 예측합니다.
