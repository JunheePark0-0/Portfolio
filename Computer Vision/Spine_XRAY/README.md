# 척추 질환 진단을 위한 딥러닝 기반 X-ray 분석 시스템

## 📖 프로젝트 개요

본 프로젝트는 컴퓨터 비전 기술을 활용하여 척추 X-ray 이미지를 분석하고, 척추 질환 진단을 보조하는 시스템을 개발합니다. YOLOv8 모델을 기반으로 척추뼈의 위치를 정확하게 탐지(Bounding Box)하고, 각 척추뼈의 건강 상태를 분류(Classification)하는 2단계 파이프라인을 구축하여 척추 질환 진단의 효율성과 정확도를 향상시키는 것을 목표로 합니다.

## 🕐 개발 기간 
2025.05 ~ 2025.05 (1개월)

## ✨ 주요 기능 및 핵심 프로세스

이 시스템은 두 가지 핵심 단계로 구성되어 척추 X-ray 이미지를 심층적으로 분석합니다.

### 1. 척추뼈 위치 탐지 (Bounding Box Detection)
-   **목표**: X-ray 이미지 내에서 개별 척추뼈의 정확한 위치를 식별하고 경계 상자(Bounding Box)로 표시합니다.
-   **기술**: `ultralytics` 라이브러리의 YOLOv8 객체 탐지 모델을 활용하여 척추뼈를 자동으로 찾아냅니다.
-   **의의**: 이 단계는 다음 분류 단계의 정확도를 높이는 전처리 과정이자, 의사가 특정 척추뼈에 집중하여 진단할 수 있도록 시각적인 정보를 제공합니다.

### 2. 척추뼈 건강 상태 분류 (Spinal Condition Classification)
-   **목표**: 탐지된 각 척추뼈 이미지(또는 해당 영역)를 기반으로 해당 뼈의 건강 상태를 분류합니다.
-   **기술**: YOLOv8 분류 모델을 사용하여 척추뼈가 정상인지, 혹은 특정 질환(예: 디스크 변형, 척추전방전위증 등)의 징후를 보이는지 판별합니다.
-   **의의**: 이 단계는 척추 질환의 자동 진단을 지원하여 의료진의 진단 효율성을 크게 개선하고, 객관적인 판단 근거를 제공합니다.

## 📂 프로젝트 구조

```
.
├── .gitignore
├── README.md
├── Bounding Box/
│   ├── Bounding Box_AP.ipynb   # 척추뼈 바운딩 박스 탐지 (AP 뷰)
│   └── Bounding Box_LA.ipynb   # 척추뼈 바운딩 박스 탐지 (LA 뷰)
├── Classification/
│   ├── Classification_AP.ipynb # 척추뼈 건강 상태 분류 (AP 뷰)
│   └── Classification_LA.ipynb # 척추뼈 건강 상태 분류 (LA 뷰)
├── DATA/                   # 데이터셋 저장 폴더 (X-ray 이미지 및 어노테이션)
└── Models/                 # 학습된 모델 가중치 저장 폴더
```

## 🛠️ 주요 구성 요소

-   **`Bounding Box/`**: 척추 X-ray 이미지에서 척추뼈의 위치를 탐지하는 모델 학습 및 평가 코드를 포함합니다. 주로 YOLOv8 객체 탐지 모델을 사용합니다.
-   **`Classification/`**: 탐지된 척추뼈를 입력으로 받아 각 척추뼈의 건강 상태를 분류하는 모델 학습 및 평가 코드를 포함합니다. 주로 YOLOv8 분류 모델을 사용합니다.
-   **`DATA/`**: 프로젝트에서 사용되는 척추 X-ray 이미지 데이터와 그에 대한 바운딩 박스 어노테이션 및 분류 라벨 데이터가 저장됩니다.
-   **`Models/`**: 학습이 완료된 객체 탐지 및 분류 모델의 가중치 파일들이 저장됩니다.

## 🚀 실행 방법

1.  **환경 설정**: 필요한 라이브러리(`ultralytics` 등)를 설치하고 개발 환경을 준비합니다.
    ```bash
    # 예시: ultralytics 설치
    pip install ultralytics
    ```

2.  **데이터 준비**: `DATA/` 폴더에 척추 X-ray 이미지 데이터셋과 해당 어노테이션 파일을 구성합니다.

3.  **모델 학습 (Bounding Box)**:
    - `Bounding Box` 폴더 내의 `.ipynb` 노트북을 실행하여 척추뼈 탐지 모델을 학습시킵니다.

4.  **모델 학습 (Classification)**:
    - `Classification` 폴더 내의 `.ipynb` 노트북을 실행하여 척추뼈 분류 모델을 학습시킵니다.

5.  **시스템 활용**: 학습된 모델을 사용하여 새로운 X-ray 이미지에 대한 척추 질환 진단 보조를 수행합니다.
